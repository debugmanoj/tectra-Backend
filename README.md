#  Doctor Management Backend (Node.js + Express + MongoDB)

This is the **backend API** for the **Doctor Management System**, built using **Express.js** and **MongoDB**.  
It provides RESTful endpoints to manage doctor records with **cursor-based pagination** for scalable data loading.

---

##  Features

-  CRUD operations for Doctors (Create, Read, Update, Delete)
-  Cursor-based pagination for optimized performance
-  Modular MVC architecture (Controller, Service, Repository)
-  Security using **Helmet** and **CORS**
-  Environment-based configuration
-  Fully deployed and working on **Vercel**

---

##  Tech Stack

| Layer | Technology |
|-------|-------------|
| Runtime | Node.js |
| Framework | Express.js |
| Database | MongoDB (Mongoose) |
| Security | Helmet, CORS |
| Environment Config | Dotenv |
